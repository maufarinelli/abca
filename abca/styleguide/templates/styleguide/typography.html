Typography

Base font-size

Used to calculate all others units of ABCA, as we use rem for font-sizes, etc.

            
    h1, .h1 {
        font-size: 3.6rem; // 36px for 10px font base // 43.2px for 12px font base
    }

    h2, .h2 {
        font-size: 3.0rem; // 30px for 10px font base // 36px for 12px font base
    }
